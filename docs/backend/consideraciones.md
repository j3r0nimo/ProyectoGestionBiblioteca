### Elementos a considerar

El proyecto es un demostrador inicial de posibilidades de diseño de una API REST, en este caso a partir de una biblioteca personal, en Node.js y con javascript.

### Posibilidades

La intención final debe definirse claramente, por cuanto es posible ingresar títulos y texto relacionado con el contenido de las publicaciones, pero deberían proveer indicios claros y concretos del contenido, para facilitar la búsqueda y localización de libros y contenido de los mismos.

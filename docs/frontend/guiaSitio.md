# Guia de uso del sitio

## Pagina principal (Libreria)

- ### Busqueda

  - al ingresar los datos del libro, este va a aparecer tras tocar **buscar**
  - ![busqueda](/IMG/busqueda.png)

- ### Listado

  - listado de todos los libros
  - ![listado](/IMG/listado.png)

- ### Paginacion

  - botones que permiter paginacion y se deshabilitan al llegar a la pagina limite
  - ![paginacion](/IMG/paginacion.png)

- ### Seleccion de formulario

  - 3 botones en la pagina **añadir** que generan formularios para la creacion de libros
  - ![seleccion](/IMG/seleccionForm.png)

- ### Creacion de libro

  - Formulario que toma todos los datos del libro y al apretar **añadir** lo sube a la base de datos
  - ![form](/IMG/form.png)

- ### Detalles de Libro

  - pagina que detalla todos los datos del libro junto su contenido y la opcion de borrar y editar dicho libro
  - ![libroEspecifico](/IMG/libroEspecifico.png)

- ### Detalles de Libro

  - pagina que detalla todos los datos del libro junto su contenido y la opcion de borrar y editar dicho libro
  - ![libroEspecifico](/IMG/libroEspecifico.png)

- ### Edicion de Libro

  - Formulario para el **Update** del libro
  - ![formUpdate](/IMG/formUpdate.png)

- ### Creacion de contenido

  - Formulario dentro del libro que permite la adicion de contenidos
  - ![formContenido](/IMG/formContenido.png)

- ### Contenidos

  - Todos los contenidos del libro, cada uno con la opcion de modificarlos o borrarlos
  - ![contenidos](/IMG/contenidos.png)

- ### Modificacion de contenidos

  - Formulario que toma los datos del contenido seleccionado y permite el **Update** del mismo
  - ![updateContenido](/IMG/updateContenido.png)

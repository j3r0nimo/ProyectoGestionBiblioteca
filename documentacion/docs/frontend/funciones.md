# Funciones

## Características Principales

* **Gestión de Publicaciones (CRUD):**
  * **Crear (POST):** Un formulario dinámico que te permite añadir nuevos Libros, Revistas u Otros (Manuales, Catálogos, Folletos), tambien permite la subida de una imagen de portada.
  * **Leer (GET):** Muestra libros, revistas, etc. que estan almacenados en la base de datos utilizando paginación.
  * **Actualizar (PUT):** Permite editar la información de una publicación existente.
  * **Eliminar (DELETE):** Permite borrar una publicación desde su página de detalles.
* **Gestión de Contenido:** Dentro de la página de detalles de una publicación, se puede gestionar un listado de "Contenidos" o "Artículos" asociados (título, resumen, página), el cual permite crear, editar y eliminar artículos específicos para esa publicación.
* **Búsqueda:** Incluye una barra de búsqueda para filtrar por palabra clave.

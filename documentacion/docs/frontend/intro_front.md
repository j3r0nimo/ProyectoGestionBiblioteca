# Introducción al Frontend

Esta sección explica la estructura del proyecto de frontend, componentes y flujo de datos.

## Instalacion y Arranque:

Para la instalacion de las dependencias del Frontend (Axios, React y React Router) y arranque del Frontend vamos a usar la terminal:

- `cd frontend` para ubicarnos en la carpeta frontend.
- `npm install` para instalar las dependencias.
- `npm run dev` para hacer correr el frontend.
- va a aparecer en la terminal el url `http://localhost:5173/` significando que cargo el Frontend y se puede visualizar.
